# /mod-ios - iOS Modding & Bypass Engine

Lệnh này kích hoạt trạng thái **Super Modder** bằng cách hội quân toàn bộ Specialist và Skills liên quan đến iOS Modding.

## Cách sử dụng:

`/mod-ios [Yêu cầu của bạn]`

## Quy trình tự động:

1. **Triệu hồi Agent:** `@[ios-mod-specialist]`
2. **Nạp dữ liệu:** Quét file IPA/Binary bạn cung cấp.
3. **Phân tích:** Dùng `@ios-modding-decision-tree` để tìm chiến thuật Mod tốt nhất.
4. **Thực thi:** Đưa ra Script Frida hoặc mã ASM cần Patch.

---

**Ví dụ:**

- `/mod-ios: Hướng dẫn tôi loại bỏ quảng cáo trong app này.`
- `/mod-ios: Tại sao đoạn code hook của tôi làm app bị crash khi khởi động?`
